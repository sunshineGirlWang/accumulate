此项目是利用rem来实现响应式的H5新闻首页的项目。

说明：

1、使用了sass编写css，webpack编译文件。

2、在开发过程中，使用webpack --watch命令实现热更新。

3、利用px2rem方法实现将px转化成rem的功能。
